# LA7_Marisa Muharomah - Penjelasan ACT

## Pertemuan 1 (rpl-pertemuan1)
ACT1: Konsep Dasar Desain Perangkat Lunak & Arsitektur Aplikasi.
Fungsi utama dari ACT1 adalah untuk memahami konsep fundamental dalam Desain Perangkat Lunak.

## Pertemuan 2 (rpl2)
ACT2: Konsep Dasar Object-Oriented Programming (OOP).
Fungsi utama dari ACT2 adalah untuk memahami dan menerapkan konsep inti dari Object-Oriented Programming (OOP) di Java.

## Pertemuan 3 (pert3-rpl) 
ACT3: Koneksi Database Java Biasa (JDBC).
Fungsi utama dari ACT3 adalah untuk membuat dan mengelola koneksi dasar ke database menggunakan Java Database Connectivity (JDBC)

## Pertemuan 4 (rpl_pert4)
ACT4: Konsep Dasar ORM dan Framework Hibernate.
Fungsi utama dari ACT4 adalah untuk mengimplementasikan operasi database menggunakan Object Relational Mapping (ORM) dengan framework Hibernate.

## Pertemuan 5 (pert5_frspring)
ACT5: Dependency Injection pada Spring Boot.
Fungsi utama dari ACT5 adalah untuk memahami dan mengimplementasikan Dependency Injection (DI) menggunakan Spring Boot untuk aplikasi konsol.

## Pertemuan 6 (rpl_pert6)
ACT6: Integrasi Database dengan GUI (Swing) pada Spring Boot.
Fungsi utama dari ACT6 adalah untuk mengintegrasikan layanan database Spring Boot dengan Graphical User Interface (GUI) menggunakan Java Swing.
